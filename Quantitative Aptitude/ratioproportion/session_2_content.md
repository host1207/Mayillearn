# Ratio & Proportion - Part 2
## Advanced Patterns & Fractional Ratios

In this session, we dive deeper into advanced ratio patterns, including vertical multiplication (VM method) for 3+ variables, handling fractions in ratios, and algebraic substitution.

---

### 📝 Example Questions & Solutions

---

<div class="question">Q1) If A : B : C = 2 : 3 : 4, find A/B : B/C : C/A <span class="type">(Inverse Variations)</span></div>

<div class="step"><strong>Step 1:</strong> Substitute the values: A=2, B=3, C=4</div>
<div class="detail">A/B = 2/3</div>
<div class="detail">B/C = 3/4</div>
<div class="detail">C/A = 4/2 = 2</div>

<div class="step"><strong>Step 2:</strong> Form the ratio → 2/3 : 3/4 : 2/1</div>

<div class="step"><strong>Step 3:</strong> Find LCM of denominators (3, 4, 1) = 12</div>

<div class="step"><strong>Step 4:</strong> Multiply each term by 12</div>
<div class="detail">(2/3) × 12 = 8</div>
<div class="detail">(3/4) × 12 = 9</div>
<div class="detail">(2) × 12 = 24</div>

<div class="answer">Answer: 8 : 9 : 24</div>

---

<div class="question">Q2) If A : B = 2 : 3 and B : C = 4 : 5, find A : B : C <span class="type">(Vertical Multiplication)</span></div>

<div class="step"><strong>Method:</strong> Make B common</div>
<div class="detail">A : B = 2 : 3 → multiply by 4 → 8 : 12</div>
<div class="detail">B : C = 4 : 5 → multiply by 3 → 12 : 15</div>

<div class="answer">Answer: 8 : 12 : 15</div>

---

<div class="question">Q3) If A : B = 1/2 : 1/3 and B : C = 1/2 : 1/3, find A : B : C <span class="type">(Fractional Ratios)</span></div>

<div class="step"><strong>Step 1:</strong> Simplify A:B</div>
<div class="detail">1/2 : 1/3 → Cross multiply → 3 : 2</div>

<div class="step"><strong>Step 2:</strong> Simplify B:C</div>
<div class="detail">1/2 : 1/3 → 3 : 2</div>

<div class="step"><strong>Step 3:</strong> Combine A:B (3:2) and B:C (3:2)</div>
<div class="detail">B is 2 in first, 3 in second. LCM = 6</div>
<div class="detail">A:B = 9:6</div>
<div class="detail">B:C = 6:4</div>

<div class="answer">Answer: 9 : 6 : 4</div>

---

<div class="question">Q4) If a : b = 5 : 7 and c : d = 2a : 3b, find ac : bd <span class="type">(Algebra Variables)</span></div>

<div class="step"><strong>Given:</strong> a/b = 5/7 and c/d = 2a/3b</div>

<div class="step"><strong>Target:</strong> Find ac/bd = (a/b) × (c/d)</div>

<div class="step"><strong>Substitute:</strong> (a/b) × (2a/3b) = (2/3) × (a/b)²</div>

<div class="step"><strong>Value:</strong> (2/3) × (5/7)² = (2/3) × (25/49) = 50/147</div>

<div class="answer">Answer: 50 : 147</div>

---

<div class="question">Q5) If 3A = 5B and 4B = 6C, find A : C <span class="type">(Equalities to Ratios)</span></div>

<div class="step"><strong>Step 1:</strong> Find individual ratios</div>
<div class="detail">3A = 5B → A/B = 5/3</div>
<div class="detail">4B = 6C → B/C = 6/4 = 3/2</div>

<div class="step"><strong>Step 2:</strong> Combine using Chain Rule</div>
<div class="detail">A/C = (A/B) × (B/C)</div>
<div class="detail">A/C = (5/3) × (3/2) = 5/2</div>

<div class="answer">Answer: 5 : 2</div>

---

<div class="question">Q6) If a : b : c = 5 : 7 : 8, find (a + b + c) / a <span class="type">(Algebraic Substitution)</span></div>

<div class="step"><strong>Method:</strong> Direct substitution (since degree is same in numerator/denominator)</div>
<div class="detail">a=5, b=7, c=8</div>

<div class="step"><strong>Calculation:</strong> (5 + 7 + 8) / 5 = 20/5 = 4</div>

<div class="answer">Answer: 4</div>

---

<div class="question">Q7) If x : y = 6 : 5, find (x² + y²) / (x² − y²) <span class="type">(Componendo & Dividendo)</span></div>

<div class="step"><strong>Substitute:</strong> x=6, y=5</div>

<div class="step"><strong>Numerator:</strong> 6² + 5² = 36 + 25 = 61</div>

<div class="step"><strong>Denominator:</strong> 6² − 5² = 36 − 25 = 11</div>

<div class="answer">Answer: 61/11</div>

---

<div class="question">Q8) If x : y = 8 : 9, find (5x − 4y) : (3x + 2y) <span class="type">(Linear Substitution)</span></div>

<div class="step"><strong>Substitute:</strong> x=8, y=9</div>

<div class="step"><strong>Term 1:</strong> 5(8) − 4(9) = 40 − 36 = 4</div>

<div class="step"><strong>Term 2:</strong> 3(8) + 2(9) = 24 + 18 = 42</div>

<div class="step"><strong>Ratio:</strong> 4 : 42 → simplify → 2 : 21</div>

<div class="answer">Answer: 2 : 21</div>

---

<div class="question">Q9) If x / 2y = 5 / 4, find (x − y) / (x + y) + 1/9 <span class="type">(Complex Value Finding)</span></div>

<div class="step"><strong>Step 1:</strong> Find x:y</div>
<div class="detail">x/2y = 5/4 → x/y = 10/4 = 5/2</div>
<div class="detail">So x=5, y=2</div>

<div class="step"><strong>Step 2:</strong> Solve bracket part</div>
<div class="detail">(x−y)/(x+y) = (5−2)/(5+2) = 3/7</div>

<div class="step"><strong>Step 3:</strong> Add 1/9</div>
<div class="detail">3/7 + 1/9 = (27 + 7)/63 = 34/63</div>

<div class="answer">Answer: 34/63</div>

---

<div class="question">Q10) If a : b = 2 : 3 and b : c = 4 : 5, find a² : b² : bc <span class="type">(Square Ratios)</span></div>

<div class="step"><strong>Step 1:</strong> Combine a:b:c</div>
<div class="detail">a:b = 2:3 → multiply by 4 → 8:12</div>
<div class="detail">b:c = 4:5 → multiply by 3 → 12:15</div>
<div class="detail">a:b:c = 8:12:15</div>

<div class="step"><strong>Step 2:</strong> Calculate terms</div>
<div class="detail">a² = 8² = 64</div>
<div class="detail">b² = 12² = 144</div>
<div class="detail">bc = 12 × 15 = 180</div>

<div class="step"><strong>Step 3:</strong> Simplify 64 : 144 : 180</div>
<div class="detail">Divide by 4 → 16 : 36 : 45</div>

<div class="answer">Answer: 16 : 36 : 45</div>

---

<div class="question">Q11) If A:B=2:3, B:C=4:5, C:D=6:7, find A:B:C:D <span class="type">(Four Variable Chain)</span></div>

<div class="step"><strong>Step 1:</strong> Combine A:B:C first</div>
<div class="detail">A:B:C = 8:12:15</div>

<div class="step"><strong>Step 2:</strong> Connect C:D (6:7)</div>
<div class="detail">Existing C = 15, New C = 6</div>
<div class="detail">LCM(15, 6) = 30</div>
<div class="detail">Multiply A:B:C by 2 → 16:24:30</div>
<div class="detail">Multiply C:D by 5 → 30:35</div>

<div class="answer">Answer: 16 : 24 : 30 : 35</div>

---

<div class="question">Q12) If a:b = 2/9 : 1/3, b:c = 2/7 : 5/7, c:d = 6:7, find a:b:c:d <span class="type">(Complex Fraction Chain)</span></div>

<div class="step"><strong>Step 1:</strong> Simplify fractions</div>
<div class="detail">a:b = 2/9 : 3/9 = 2:3</div>
<div class="detail">b:c = 2:5</div>
<div class="detail">c:d = 6:7</div>

<div class="step"><strong>Step 2:</strong> Combine a:b:c</div>
<div class="detail">Make b common (LCM of 3 and 2 = 6)</div>
<div class="detail">a:b = 4:6</div>
<div class="detail">b:c = 6:15</div>
<div class="detail">a:b:c = 4:6:15</div>

<div class="step"><strong>Step 3:</strong> Join d</div>
<div class="detail">c=15, next c=6. LCM=30</div>
<div class="detail">Multiply a:b:c by 2 → 8:12:30</div>
<div class="detail">Multiply c:d by 5 → 30:35</div>

<div class="answer">Answer: 8 : 12 : 30 : 35</div>
