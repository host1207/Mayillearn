## Markdown Cheat Sheet

Welcome! This is your quick reference guide for writing MayilQ study notes. Everything below is a **live demo** - what you see is what the formatting looks like!

---

## Headings

# This is Heading 1 (Biggest)
## This is Heading 2 (Section Title)
### This is Heading 3 (Sub-section)
#### This is Heading 4 (Detail)

---

## Text Styling

**This text is bold** - use for important terms

*This text is italic* - use for emphasis

`This is code/formula text` - use for formulas like `Speed = Distance / Time`

---

## Colours

{red}This text is RED{/red} - use for warnings or wrong answers

{green}This text is GREEN{/green} - use for correct answers

{blue}This text is BLUE{/blue} - use for definitions

{yellow}This text is YELLOW{/yellow} - use for highlights

{orange}This text is ORANGE{/orange} - use for caution

{purple}This text is PURPLE{/purple} - use for special notes

{cyan}This text is CYAN{/cyan} - use for formulas

{highlight}This has a glowing background{/highlight} - use for KEY formulas

---

## How to Type Colours in Obsidian

Just wrap your text like this in your .md file:

`{red}your text here{/red}`

`{green}your text here{/green}`

`{highlight}your text here{/highlight}`

## Images

Images show at default size (420px). You can control the size:

- Default: `![name](url)`
- Small: `![name|small](url)`
- Large: `![name|large](url)`
- Custom: `![name|300px](url)`

## Inline Quiz Questions

You can embed quiz questions ANYWHERE in your notes! They appear as interactive widgets with green/red feedback.

```quiz
question: What symbol makes text bold in Markdown?
A: ##text##
B: **text**
C: __text__
D: ~~text~~
answer: B
explanation: Two asterisks on each side **like this** makes text bold!
```

### Another Example

The formula for combined work rate is `1/a + 1/b`.

```quiz
question: What colour tag would you use to mark a CORRECT answer?
A: {red}
B: {blue}
C: {green}
D: {yellow}
answer: C
explanation: Green is universally used for correct answers! Type {green}correct{/green} in your notes.
```
